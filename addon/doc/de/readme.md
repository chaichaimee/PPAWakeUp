## 🌟 Erleben Sie nahtlose Kontrolle

Haben Sie genug von Sprachunterbrechungen? **PPAWakeUp** ist das revolutionäre NVDA-Add-on, das die Steuerung der **PPA Tatip** Thai-Stimme neu definiert. Mit unserer **Smart-Tap-Technologie** beherrschen Sie Ihren Synthesizer mit nur einer Tastenkombination – blitzschnell und effizient.

---

## ⚡ Die Macht von "Windows + P"
*Eine Taste. Drei Aktionen. Absoluter Komfort.*

| Geste | Aktion | Beschreibung |
| :--- | :--- | :--- |
| **Einmal tippen** | 🚀 **Sofortige Wiederbelebung!** | Startet `windows_tatip.exe` neu. Falls die Stimme hängt, ist sie sofort wieder da! |
| **Zweimal tippen** | ⚙️ **Präzise Einstellungen** | Springen Sie direkt in die PPA Tatip Optionen für schnelle Anpassungen. |
| **Dreimal tippen** | 📖 **Wörterbuch-Master** | Direkter Zugriff auf `userdict.txt`, um die Aussprache sofort zu bearbeiten. |

> [!TIP]
> **Profi-Tipp:** Tippen Sie einfach 1, 2 oder 3 Mal kurz hintereinander. Der intelligente Algorithmus erledigt den Rest.
