PPAWakeUp

Раскройте весь потенциал PPA Tatip с помощью одной умной клавиши!

Безупречный контроль

Надоели сбои в озвучке? PPAWakeUp — это революционное дополнение для NVDA, которое меняет ваш опыт работы с тайским синтезатором PPA Tatip.

Windows+P (1 нажатие) → Мгновенное оживление! Перезапуск процесса при зависании голоса.
Windows+P (2 нажатия) → Точная настройка Быстрый переход к параметрам синтезатора.
Windows+P (3 нажатия) → Работа со словарем Прямой доступ к редактированию пользовательского словаря.
