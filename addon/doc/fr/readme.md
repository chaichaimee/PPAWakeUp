## 🌟 Un contrôle fluide et rapide

Marre des interruptions vocales ? **PPAWakeUp** est l'extension NVDA révolutionnaire qui transforme votre interaction avec le moteur de synthèse **PPA Tatip**. Grâce à notre technologie **Smart-Tap**, commandez votre voix d'un seul geste.

---

## ⚡ La puissance de "Windows + P"
*Une touche. Trois actions. Une simplicité absolue.*

| Gestes | Action | Description |
| :--- | :--- | :--- |
| **Une pression** | 🚀 **Réveil instantané !** | Relance `windows_tatip.exe` immédiatement si la voix se fige. |
| **Deux pressions** | ⚙️ **Réglages de précision** | Accédez instantanément aux options et réglages de PPA Tatip. |
| **Trois pressions** | 📖 **Maîtrise du dictionnaire** | Ouverture directe de `userdict.txt` pour corriger la prononciation. |

---

## 🔥 Pourquoi choisir PPAWakeUp ?

* **Zéro Complexité :** Une touche unique pour tout contrôler.
* **Ultra-Léger :** Consommation de ressources système quasi inexistante.
* **Conçu pour les Pros :** Spécialement optimisé pour les utilisateurs de PPA Tatip Thai.
