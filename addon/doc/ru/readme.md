## 🌟 Безупречный контроль

Надоели сбои в озвучке? **PPAWakeUp** — это революционное дополнение для NVDA, которое меняет ваш опыт работы с тайским синтезатором **PPA Tatip**. Мы переосмыслили эффективность, предоставив вам полный контроль над голосом с помощью одной комбинации.

---

## ⚡ Магия "Windows + P"
*Одна клавиша. Три действия. Абсолютное удобство.*

| Жест | Действие | Описание |
| :--- | :--- | :--- |
| **Одно нажатие** | 🚀 **Мгновенное оживление!** | Перезапуск `windows_tatip.exe`. Если голос пропал — верните его за секунду! |
| **Два нажатия** | ⚙️ **Точная настройка** | Быстрый переход к параметрам и настройкам PPA Tatip. |
| **Три нажатия** | 📖 **Работа со словарем** | Прямой доступ к файлу `userdict.txt` для правки произношения. |

---

## 🔥 Почему PPAWakeUp — это маст-хэв?

* **Никакой сложности:** Одна горячая клавиша для всех задач. Просто и чисто.
* **Минимальные ресурсы:** Оптимизированный код, который не грузит систему.
* **Для профессионалов:** Разработано специально для активных пользователей PPA Tatip.
