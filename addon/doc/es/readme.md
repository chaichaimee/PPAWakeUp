## 🌟 Control sin interrupciones

¿Cansado de que la voz se detenga? **PPAWakeUp** es el complemento revolucionario para NVDA que transforma tu interacción con el motor de voz **PPA Tatip**. Hemos redefinido la eficiencia para que puedas gestionar tu sintetizador sin perder un segundo.

---

## ⚡ El poder de "Windows + P"
*Una tecla. Tres acciones. Comodidad absoluta.*

| Gestos | Acción | Descripción |
| :--- | :--- | :--- |
| **Un toque** | 🚀 **¡Reanimación instantánea!** | Reinicia `windows_tatip.exe` al instante si la voz se bloquea. |
| **Dos toques** | ⚙️ **Ajustes de precisión** | Salta directamente a las opciones de configuración de PPA Tatip. |
| **Tres toques** | 📖 **Maestro del diccionario** | Acceso directo a `userdict.txt` para editar la pronunciación sobre la marcha. |
